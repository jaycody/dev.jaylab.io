What can js do?

event handlers can handle and verify user input, user actions, and browser actions, i.e.:

  • trigger every time a page loads
  • trigger upon page close
  • trigger upon button click
  • verify user input

methods by which js can work with events

  • HTML event attributes can execute js code directly
  • HTML event attributes can call js functions
  • create and assign custom event handler functions to HTML elements
  • PREVENT events from being sent or from being handled

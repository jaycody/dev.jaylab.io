Document Object Model

  • the topmost object in the browser object model (BOM) is the window
  • the topmost object in the DOM is the document
  • the document object of the DOM represents the web page loaded into the current browser or tab
  • the DOM is vital to accessing and amending the contents of the current page

properties of the DOM:

  • document.title
  • document.lastModified
  • document.URL
  • document.domain

methods of the DOM:

DOM METHOD DESCRIPTION
document.write() writes text to document
document.getElementById() returns element, if there is an element whose id attribute matches the argument
document.querySelectorAll() returns a list of elements that match CSS selector passed as an argument
document.createElement() creates new element
document.createTextNode() creates new text node. Really?

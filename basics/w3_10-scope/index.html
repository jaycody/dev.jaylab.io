js_gungfu-w3_tutorials-10-scope

  • scope is the set variables, objects, and functions you have access to
  • js has FUNCTION SCOPE: the scope changes inside functions
  • vars declared inside a function have local scope and are created/destroyed when the function starts and ends
  • vars declared outside a function become GLOBALS. all scripts and functions can access it
  • AUTOMATIC GLOBALS:
  • - a variable becomes global, (regardless of where it's created) if the var is undeclared when a value is assigned to it
  • - example of local var created inside function foo: var car = "datsun"
  • - example of GLOBAL var created inside function foo: car = "datsun"
  • VARIABLE LIFETIMES
  • - var lifetimes begin when vars are declared. locals are deleted when function is completed. globals are deleted when the page is closed.
  • GLOBALS in HTML
  • - in HTML, the global scope is the window object: all global variables belong to the window object
  • globals and functions can overwrite window variables
  • any function, including a window object, can overwirte global variables and functions
